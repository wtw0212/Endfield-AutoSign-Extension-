# SKPORT Auto Sign-in Extension

此 Chrome 擴充功能用於自動執行 SKPORT 上《明日方舟：終末地》與《明日方舟》的每日簽到任務。
Chrome store link:
https://chromewebstore.google.com/detail/Endfield%20%E8%87%AA%E5%8B%95%E7%B0%BD%E5%88%B0/dlfggcgclahjbigdebgfamllaajjfimj

## Description / 簡介

**[English]**
A Chrome extension for SKPORT daily sign-ins. It automatically completes daily check-ins for Arknights: Endfield and Arknights, ensuring you never miss a reward. Features include scheduled checks, automatic page navigation, and one-click manual sign-in, with full support for both Traditional Chinese and English.

**[繁體中文]**
這是一個專為 SKPORT 每日簽到設計的 Chrome 擴充功能。它能每天自動為您完成《明日方舟：終末地》與《明日方舟》官網簽到，確保您不會錯過任何獎勵。具備定時檢查、自動展開頁面與一鍵手動簽到功能，並完全支援繁體中文與英文介面。

## 功能

- 📅 **定時檢查**：每天凌晨 00:10 自動檢查簽到狀態。
- 🔓 **無人值守**：自動打開 Endfield 與明日方舟簽到頁面、展開日曆並點擊簽到。
- ✅ **網站開關**：可在歡迎頁或 Popup 選擇啟用 Endfield、明日方舟，或兩者都啟用。
- 🔔 **狀態即時顯示**：Popup 介面分別顯示兩個遊戲的上次簽到日期與目前狀態。
- ⚡ **手動觸發**：可透過擴充功能圖示隨時手動啟動全部支援遊戲的簽到流程。
- 🌐 **多語言支援**：支援繁體中文與英文。
- 🆕 **安裝/更新頁**：安裝時可直接設定網站與簽到時間；略過時會使用預設 00:10。

## 注意事項
- 記得先開啓瀏覽器「執行背景應用程式」的權限，自動開啓瀏覽器的功能才會生效。
- 執行簽到時請確保瀏覽器已登入 SKPORT 帳號。
- 簽到完成後，自動簽到分頁會在背景關閉；若簽到失敗，分頁會保持開啟以便確認狀態。
