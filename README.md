# Endfield Auto Sign-in Extension

此 Chrome 擴充功能用於自動執行《明日方舟：終末地》官網的每日簽到任務。
Chrome store link:

## Description / 簡介

**[English]**
A Chrome extension specifically designed for Arknights: Endfield players. It automatically completes your daily check-in on the official website, ensuring you never miss a reward. Features include scheduled checks, automatic page navigation, and one-click manual sign-in, with full support for both Traditional Chinese and English.

**[繁體中文]**
這是一個專為《明日方舟：終末地》玩家設計的 Chrome 擴充功能。它能每天自動為您完成官網簽到，確保您不會錯過任何獎勵。具備定時檢查、自動展開頁面與一鍵手動簽到功能，並完全支援繁體中文與英文介面。

## 功能

- 📅 **定時檢查**：每天凌晨 04:10 自動檢查簽到狀態。
- 🔓 **無人值守**：自動打開簽到頁面、展開日曆並點擊簽到。
- 🔔 **狀態即時顯示**：Popup 介面顯示上次簽到日期與目前狀態。
- ⚡ **手動觸發**：可透過擴充功能圖示隨時手動啟動簽到流程。
- 🌐 **多語言支援**：支援繁體中文與英文。

## 安裝方式

1. 下載本專案原始碼。
2. 開啟 Chrome 瀏覽器，進入 `chrome://extensions/`。
3. 開啟右上角的「開發者模式」。
4. 點擊「載入解壓縮產物」，並選擇 `endfield-auto-sign-extension` 資料夾。

## 如何使用

- **自動模式**：擴充功能會自動在後台運行。每天 04:10 如果偵測到尚未簽到，會自動開啟分頁執行。
- **手動模式**：點擊瀏覽器工具列的擴充功能圖示，點選「立即手動簽到」。

## 注意事項

- 執行簽到時請確保瀏覽器已登入 SKPORT 帳號。
- 簽到完成後，頁面會保持開啟狀態以便確認結果。
